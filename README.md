# Airbnb Amsterdam ETL Pipeline

Real-world data engineering project processing Inside Airbnb listings, reviews and calendar data for Amsterdam.

## Goal
- Learn production-grade ETL practices
- Handle 100 MB → 1 GB+ data
- Prepare portfolio piece for HSM-sponsored roles in the Netherlands

## Tech stack
Python • Pandas • SQLite (local) → BigQuery (cloud) • Logging • Git • GCP free tier
